# Cross-Border Product Research

**跨境电商选品调研深度工具** — 找到「能卖且赚钱」的产品。

## 为什么这个技能值钱

选品是跨境电商最关键的环节。**选对产品 = 成功80%**。
一个正确的选品决策能带来 6-12 个月的稳定利润。
专业选品顾问收费 ¥5000-20000/次，这个技能把选品流程工具化。

## 核心工作流

### 第一阶段：市场扫描（找出候选类目）

**输入条件：**
```
客单价: $15-50
月销量: >300单
竞争: 头部品牌市占率<30%
趋势: 逐月上升或稳定
物流: 小件轻便（FBA费用<$6）
```

**数据源扫描：**

| 渠道 | 查什么 | 工具 |
|------|--------|------|
| Amazon BSR | 类目TOP100销量分布 | Keepa/卖家精灵 |
| Google Trends | 搜索趋势是否上升 | Google Trends API |
| 社交媒体 | TikTok/Instagram热度 | 爬虫/手动 |
| 1688/阿里国际 | 供应商数量和价格 | 1688搜索 |
| 海关数据 | 进出口趋势 | 海关信息网 |

### 第二阶段：竞争分析（判断能不能做）

**品牌集中度分析：**
```
类目: Wireless Earbuds
          ↓
头部3品牌市占率: 62% ❌ (太集中，不建议进入)
类目: Pet Hair Remover
头部3品牌市占率: 18% ✅ (分散，有机会)
```

**Review 壁垒判断：**
```
类目前10平均Review: 2,500条  ❌ (>1000条，壁垒高)
类目前10平均Review: 200条   ✅ (<500条，可以打)
新品3个月内进前50: 有2个    ✅ (新品有机会)
```

**价格带分析：**
```
$5-10:  15个产品 (低价走量)
$10-20: 42个产品 (主流价格带) ← 你选择的区间
$20-30: 28个产品
$30+:   8个产品
→ 你选的价格带竞争适中，有利润空间
```

### 第三阶段：利润精算（算清楚赚多少）

```
产品售价:           $19.99
亚马逊佣金(15%):    -$3.00
FBA物流费:          -$4.50
产品成本(1688):     -$3.00
头程运费:           -$1.20
广告费(ACoS20%):    -$4.00
退货损耗(5%):       -$1.00
─────────────────────────
单品净利润:          $3.29  (16.5%净利率)
月销300单:          $987/月
月销1000单:         $3,290/月
```

### 第四阶段：供应商验证

```
1688供应商: 深圳市XX电子有限公司
          ↓
✅ 工厂实拍：有（企业认证）
✅ 资质证书：CE, FCC, RoHS
✅ MOQ: 100pcs ✓
✅ 样品: 可打样（¥50+运费）
✅ 交期: 7-15天
✅ 评价: 4.8星，98%好评
✅ 响应: <5分钟
─────────────────────────
结论: 可靠供应商，建议下样品单
```

